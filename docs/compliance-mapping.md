# Compliance Mapping

This framework supports evidence collection and operational controls
commonly required by security and compliance standards.

## NIST Cybersecurity Framework
- ID.AM: Asset health visibility
- PR.IP: Secure configuration and maintenance
- DE.CM: Continuous monitoring
- RS.MI: Mitigation through controlled remediation

## SOC 2
- Change management controls
- System availability assurance
- Logged, repeatable operational procedures

## Operational Benefits
- Improved patch success rates
- Reduced rollback incidents
- Clear audit trails for system health
- Defensible remediation workflows
